# Express Boilerplate

## 환경
* Node 12.1
* Express 
* Sequelize
* MySQL 8
* Redis 5

## Docker
```
docker-compose -f docker-compose.yml up
```

## 폴더 구조
```
.
+-- config            
+-- database          
  +-- model.loader
  +-- index           
  +-- models          
+-- domain            
+-- http              
+-- security          
+-- utils             
+-- test              
+-- index.js          
```
